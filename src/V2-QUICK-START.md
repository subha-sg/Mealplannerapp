**Sections (Top to Bottom):**
1. Dietary Preference
2. Nutrition Focus  
3. Health Focus
4. Prep Time
5. Meal Type

**How Filters Work:**
- Multiple selections = OR logic within category
- Multiple categories = AND logic between categories
- Example: (Veg OR NonVeg) AND (HighProtein) AND (Breakfast)
